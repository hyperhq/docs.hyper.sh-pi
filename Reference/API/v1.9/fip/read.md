Read
---------------------------------
read a floating IPs

### HTTP Request

`GET /api/v1/hyper/fips/{ip}`

### Path Parameters

| Parameter | Description |
| --- | --- |
| ip | The floating IP to set |

### Response

| Code | Description |
| --- | --- |
| 200 _[FipResponse](index.md#fipresponse) array_ | OK |
| 400 | Bad Request |
| 404 | Not Found |
| 500 | Server Error |

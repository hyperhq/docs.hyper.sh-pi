Read
---------------------------------
read the specified volume

### HTTP Request

`GET /api/v1/hyper/volumes/{name}`

### Query Parameters

| Parameter | Description |
| --- | --- |
| name | The volume name |

### Response

| Code | Description |
| --- | --- |
| 200 _[VolumeResponse](index.md#volumeresponse)_ | OK |
| 400 | Bad Request |
| 404 | Not Found |
| 500 | Server Error |

}
